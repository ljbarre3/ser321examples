You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=2&num2=3 -- returns the result of 2 and 3 multiplied together.
  • /dateDifference?date1=20240215&date2=20190930 -- returns the dates in MM/DD/YYYY format and tells you how far apart the two dates are.
    • Please Make sure the dates are in YYYYMMDD format
  • /stringConcatenate?string1=string&string2=Concatenate -- returns a concatenated version of both strings
